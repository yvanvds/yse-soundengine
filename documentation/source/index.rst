libYSE
======

libYSE is a cross-platform C++ sound engine. Version 2.0 removes the JUCE
dependency; PortAudio and libsndfile are the only audio backends. Windows,
Linux, and Android are supported.

This site documents the public C++ API. Build and contributor docs live in
the repository ``README.md``.
